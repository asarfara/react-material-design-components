Flat Button

```js
<Button>Default Button</Button>

<Button ripple="false">Ripple Off Button</Button>

<Button additionalClassNames="button--primary">Primary Button</Button>

<Button additionalClassNames="button--secondary">Secondary Button</Button>

<Button disabled>Disabled Button</Button>
```

Raised Button

```js
<Button additionalClassNames="button--raised">Default Raised Button</Button>

<Button additionalClassNames="button--raised" ripple="false">Ripple Off Button</Button>

<Button additionalClassNames="button--raised button--primary">Primary Raised Button</Button>

<Button additionalClassNames="button--raised button--secondary">Secondary Raised Button</Button>

<Button additionalClassNames="button--raised" disabled>Disabled Button</Button>
```

Flat Links

```js
<Button href="#default">Default Link</Button>

<Button href="#default" ripple="false">Ripple Off Link</Button>

<Button href="#default" additionalClassNames="button--primary">Primary Link</Button>

<Button href="#default" additionalClassNames="button--secondary">Secondary Link</Button>

<Button href="#default" disabled>Disabled Link</Button>
```

Raised Links

```js
<Button additionalClassNames="button--raised" href="#default">Default Raised Button</Button>

<Button additionalClassNames="button--raised" href="#default" ripple="false">Ripple Off Button</Button>

<Button additionalClassNames="button--raised button--primary" href="#default">Primary Raised Button</Button>

<Button additionalClassNames="button--raised button--secondary" href="#default">Secondary Raised Button</Button>

<Button additionalClassNames="button--raised" href="#default" disabled>Disabled Button</Button>
```
